# Telegram‑бот с базой знаний об основоположниках космонавтики

Этот проект создаёт Telegram‑бота на базе библиотеки **aiogram 3**, который отвечает на вопросы пользователей, используя базу знаний, сформированную из статей Википедии о пионерах космонавтики.


## Функциональность

Бот умеет:
* Отвечать на вопросы о ранних исследователях и учёных в области космонавтики, используя контекст из базы знаний.
* Переводить запросы пользователя с русского на английский (для работы с англоязычной базой).
* Переводить ответы модели с английского на русский для удобства пользователя.
* По команде `/help` или при нажатии кнопки «Инфо» возвращать информацию о базе знаний:
    * тематику;
    * количество записей;
    * пример запроса.

## Как это работает

1. **Сбор данных:** скрипт собирает заголовки статей из категории Википедии *Early spaceflight scientists* и извлекает из них текстовые секции.
2. **Очистка и обработка:** текст очищается от ссылок и лишних пробелов, фильтруются короткие или пустые секции.
3. **Токенизация:** для каждого фрагмента текста вычисляются эмбединги (векторные представления) с помощью модели OpenAI *text‑embedding‑ada‑002*.
4. **Поиск релевантных данных:** при получении вопроса бот находит наиболее релевантные фрагменты из базы знаний по схожести эмбедингов (косинусное расстояние).
5. **Формирование ответа:** релевантные фрагменты передаются в модель GPT‑3.5‑Turbo, которая формирует итоговый ответ.
6. **Взаимодействие с пользователем:** бот принимает запросы в Telegram, переводит их, получает ответ от GPT и возвращает переведённый ответ пользователю.

## Структура проекта

* `.ipynb` — Jupyter Notebook с полным кодом проекта (сбор данных, подготовка базы знаний, создание и запуск Telegram‑бота).
* `Early_spaceflight_scientists.csv` — CSV‑файл с базой знаний (текстовые фрагменты и их эмбединги).
